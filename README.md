# SLDB
This is a database implementation to research certain aspects in the pipeline. Having all sota buffer replacement policies. In the future i try to add as many features as possible. 
# BUFFER_REPLACEMENT
- LRU
- LFU
- FIFO
- TWOQUEUE
- Clock Replacement
# INDEXING
- B-Tree
- B+-Tree
- Hashmap
# STORAGE
- row_storage (OLTP)
- column_storage (OLAP)
